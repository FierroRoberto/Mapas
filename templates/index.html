🗺️ Generador de Mapa para Google My Maps

📌 ¿Cómo usar?
- Tu Excel debe tener una columna llamada "Nombre Comercial".
- Opción A: columnas "Latitud" y "Longitud" (números).
- Opción B: columna "Dirección" (requiere API Key, configurar en backend).
- Opcional: columna "Categoría" para colorear los marcadores.
- Se incluirán TODAS las demás columnas en la ventana de información.

📂 Arrastra tu archivo Excel aquí o haz clic para seleccionar

El archivo generado (.kmz) se puede importar directamente en Google My Maps.
